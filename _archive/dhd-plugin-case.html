DHD · PLUGIN: the analysis travels, the data stays.

DHD beheert landelijke data van de Nederlandse ziekenhuizen. In het PLUGIN-project wordt analyse over ziekenhuizen heen mogelijk, federatief en zonder centrale databerg.

DHD manages national data for Dutch hospitals. The PLUGIN project enables analysis across hospitals, federated and without a central data pile.

VraagChallenge

Ziekenhuizen willen samen leren van elkaars data, maar patiëntdata mag en wil niemand zomaar centraal verzamelen.

Hospitals want to learn from each other's data, but nobody can or should simply pool patient data centrally.

AanpakApproach

Een federatieve infrastructuur op open componenten: het algoritme reist naar het ziekenhuis, rekent daar, en alleen de uitkomsten komen samen. Elke deelnemer draait dezelfde open bouwstenen.

A federated infrastructure on open components: the algorithm travels to the hospital, computes there, and only the outcomes come together. Every participant runs the same open building blocks.

ResultaatResult

Gezamenlijke inzichten over ziekenhuizen heen, terwijl de patiëntdata het eigen huis nooit verlaat. Het model is herhaalbaar voor elke volgende vraag.

Shared insight across hospitals while patient data never leaves its own house. The model is repeatable for every next question.
